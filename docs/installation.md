# Installation

## Stable release

To install Gravitational Lensing, run this command in your terminal:

```sh
uv add gravilens
```

Or if you prefer to use `pip`:

```sh
pip install gravilens
```

## From source

The source files for Gravitational Lensing can be downloaded from the [Github repo](https://github.com/GitArrenDave/GraviLens).

You can either clone the public repository:

```sh
git clone https://github.com/GitArrenDave/GraviLens
```

Or download the [tarball](https://github.com/GitArrenDave/GraviLens/tarball/main):

```sh
curl -OJL https://github.com/GitArrenDave/GraviLens/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd gravilens
uv sync
```
